Changes in CSS3

CSS3 is the latest standard for the CSS language. it also has new properties that debugs previous and extends CSS2 features.even it has javascript capability

changes in css3 are the following

Module based code

CSS3 is split into modules.

CSS3 modules include

  • Selectors
  • The box model modules
  • Background and borders
  • Text effects

Web font support gives designers access to way more than just "web safe" fonts

Web safe fonts are fonts that every single computer has installed and recognizes.

Designers can now run web fonts in css3 like google fonts and typecast.

It enables faster development and faster load times

CSS3 has visual enhancement.it saves time .pages load faster overall smaller file sizes. Because these effects are built into the code.

New colors and image effects

Css3 supports new colors such as RGBA , HSL and gradient colors Allows for adjustment to opacity. And it also supports rounded image corners.

Box sizing has fixed some annoying alignment problems

It allows to get the sizing of elements right without having to substract dimension. For padding and borders.

Why We Use Relative Size?

Relative size(%,em,rem) one styling the size of an element they do more consistency in our CSS to meet the needs of changing viewport sizes.

The most important reason for using responsive and unitless values in the CSS is for supporting the users that rely on the zooming.The users can zoom the viewport without loss of content.

  • The users shall to rezise text up to 200 percent without loss of content.
  • The users shall to resize the text without scolling horizontally and vertically.
  • Consistency on the system makes to craft agreat experience for the end of the user.

It should provide appropriate spacing between lines and paragraphs .it does not required to scroll horizontally and vertically to read in full screen.
